# nlp-project
La classification des genres musicaux est un défi qui a longtemps intéressé. Même pour les humains, la classification précise des genres musicaux peut être difficile en raison de la diversité et de la subjectivité des critères utilisés pour définir les genres. Dans notre cas, nous nous y intéressons dans le domaine du traitement du langage naturel (NLP). \
Avec l'essor des techniques d'apprentissage automatique et l'abondance des données textuelles disponibles, il est devenu possible d'explorer de nouvelles méthodes pour associer des chansons 
à leurs genres respectifs en se basant uniquement sur les paroles. \
Ce projet de groupe a pour objectif d'examiner la faisabilité de prédire le genre musical d'une musique à partir de ses paroles en utilisant des techniques avancées de NLP. Nous allons analyser différentes approches, évaluer leur performance et déterminer les caractéristiques linguistiques qui peuvent aider à discriminer les genres musicaux.
